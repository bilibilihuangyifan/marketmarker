# marketmarker

> Using grid trading strategy for market maker.

#### Run Setup

> node.js 
in js/ directory:

``` bash
# install dependencies
npm install

# run
npm run dev

```
> python
in py/ directory:

``` bash
# test
python maketmaker.py

# run
python start.py

```
